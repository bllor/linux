cli(command line interface): 명령어로 컴퓨터를 제어하는 방식
명령은 현재 머물고 있는 디렉터리에 대상으로 내려진다 --> 그러므로 현재 머물고 있는 디렉터리에 있는지 확인해야한다.
머물고 있는 디렉터리를 확인하는 방법 : pwd
디렉터리 안의 파일을 확인하는 방법 : ls 
디렉터리 안의 모든 파일을 확인하는 방법  & 파일과 디렉터리를 구분할 수 있게 도와주는 명령어: ls -al
--> -al 은 파라미터라고 불린다.
--> -rw-rw-r--과 같이 나타날 때 제일 앞에 d가 있으면 디렉터리, 없으면 파일이다.

새로운 폴더를 만드는 방법 : mkdir folder_name
비어있는 파일 만들기 : touch file_name + file확장자

디렉터리를 변경하는 방법 cd(change directory)
상위 디렉터리로 이동하는 방법 
1)pwd한다음 상위 디렉터리의 주소를 입력해줌
2)cd ../ --> ..은 현재 디렉터리의 부모디렉터리를 의미한다.
  ..은 상대적으로 달라질 수 있기 때문에 상대경로라고 한다.

pwd를 했을 때 / 인 곳이 root 디렉터리이다.

파일삭제 rm(remove) file_name --> rm으로 디렉터리를 삭제할 수 없는데, rm -r directory_name을 하여 삭제할 수 있다.

명령어 --help : 해당 명령어의 도움말이 나온다.

nano - 처음 리눅스를 사용하는 사람에게 많이 추천함
nano와 vi는 리눅스, 유닉스 계열에서 많이 사용된다.

homebrew - 프로그램을 다운로드 받는 installer 같은 역할
brew search file file을 다운 받을 수 있는지 확인할 때 사용한다.
brew list - brew로 다운받은 프로그램의 리스트를 확인할 수 있다.
brew upgrade file_name - brew로 다운 받은 파일을 업그레이드 할 수 있다. brew upgrade 만 입력하면 모든 프로그램을 업그레이드할 수 있다.
brew update - 홈브루를 통해 다운받은 프로그램을 업데이트 한다.

파일 다운로드 받기 -wget을 통해 url로 다운로드 받을 수 있다.
wget -O file_name url --> url을 통해 다운로드 받는 것의 이름을 file_name 으로 만든다.
